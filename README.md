# douyin
Just a simple app spider for douyin(Tiktok)

一个简单的抖音APP爬虫
------

###博客地址:
[https://www.weiney.com/2072.html">https://www.weiney.com/2072.html](https://www.weiney.com/2072.html">https://www.weiney.com/2072.html)

###实现功能

*appium对抖音软件的自动化控制
*mitmproxy实现对手机数据包的抓取
*百度人脸识别SDK的调用
*通过抖音uid获取真实抖音号的爬虫
