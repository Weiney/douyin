# douyin
Just a simple app spider for douyin(Tiktok)

<h2>一个简单的抖音APP爬虫</h2>

<h3>博客地址:</h3>
<a href="https://www.weiney.com/2072.html">https://www.weiney.com/2072.html</a>

<h3>实现功能<h3>
<ol>
  <li>appium对抖音软件的自动化控制</li>
  <li>mitmproxy实现对手机数据包的抓取</li>
  <li>百度人脸识别SDK的调用</li>
  <li>通过抖音uid获取真实抖音号的爬虫</li>
</ol>
